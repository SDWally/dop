## 给程序员的建议

- from https://mp.weixin.qq.com/s/myOHfJDUmqQnZohcaxiVXg 佳期某程序员

1）技术能力的提升往往来源于对日常细节问题的刨根问底；

2）持续更新自己的技术知识体系，技术深度之外也要兼顾广度；（深度在Python与数据计算，广度在服务治理，软件全周期，语言）

3）在一个能够最大化地将自己的技术转化为成果的环境中工作。
